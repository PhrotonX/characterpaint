Character Painter - by Phroton
A basic program to write and copy ASCII texts.

Requirements:
-2MB of RAM
-Windows 2000 32-bit (May not support Windows 10)

Features:
-Draw letters. INPUT->OUTPUT
-Draw with Lines, Borders, and Blocks
-Use Alphabets and numbers as additional blocks
-Design with HEX colors
-Has old encoding format (to work with borders)

Plans:
-Unused: z then [ { ] } - _ = + ! @ # $ % ^ & * ( )
